## âš™ï¸ **How the System Works**

1. Flex sensors detect finger bending and send analog values
2. Arduino reads the flex data and maps it to servo angles
3. Servo motors pull nylon tendons attached to each finger
4. The 3D-printed robotic hand mimics real-time human motion
5. Entire interaction was calibrated and tested by **pingjahangir**
